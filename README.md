TV Shows Analytics Dashboard | Power BI Project

Overview

This Power BI dashboard analyzes popular TV shows based on IMDb ratings, genres, release periods, and Netflix availability.

The project focuses on discovering trends in TV show categories and audience preferences through interactive visualizations.

---

Key Features

- IMDb Rating Analysis
- Genre Distribution
- Netflix vs Non-Netflix Comparison
- TV Shows by Release Period
- Interactive Filtering
- KPI Summary Dashboard
